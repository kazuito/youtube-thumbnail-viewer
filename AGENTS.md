# YouTube Thumbnail Viewer

This monorepo contains 2 apps in the `apps` dir.

- Chrome Extension: 'YouTube Thumbnail Viewer' web extension. It shows thumbnail image in the description area on youtube.com. Implemented on `wxt` framework.
- Website: Homepage for the extension. Uses Next.js 16 App Router.

## Instructions

- Use pnpm
- Run `pnpm typecheck` after edits
